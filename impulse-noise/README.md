### Impulse Noise Pipeline

- **`video_pipeline.py`**  
  Contains the core utilities for impulse noise handling, including filtering, subtitle mask extraction, and feeding processed frames into the deep decaptioning network to produce the final output video.

- **`impulse-noise-pipeline.ipynb`**  
  Provides an end-to-end pipeline that integrates all processing stages. Following the notebook sections gives a clear overview of how the impulse noise pipeline is structured and executed, and it is the recommended entry point for understanding the overall workflow.
